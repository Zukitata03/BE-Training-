# 0.FastAPI-Training

Production-quality FastAPI training module for the repository. This folder contains a complete FastAPI application scaffold with:
- App factory pattern, async SQLAlchemy models, Pydantic schemas, and service layer
- JWT-based authentication with access and refresh tokens
- CRUD for books with ownership checks
- Basic blockchain event indexing integration scaffold
- Celery task scaffolding and Redis-based broker
- Dockerfile and docker-compose for a full dev/stage environment
- A training guide and prompt library for agent-driven learning

Note: This is a self-contained module intended as a reference for learning production-grade backend patterns. It does not rely on external code outside this repository structure.
